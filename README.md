## 📌 프로젝트 요약

실시간 항공기 데이터를 수집해 항공사별/속도별 현황 및 궤적을 시각화하고, 일일 통계를 자동으로 분석하는 실시간 대시보드

## ☕️ 프로젝트 성격 

이 프로젝트는 데이터 엔지니어링 역량 강화를 목적으로 진행하는 **개인 스터디용 토이 프로젝트**입니다. 실시간 데이터 처리, 데이터 모델링, Asset 식별, Lineage 설계를 직접 설계·구현하며 학습하는 데 초점을 두고 있습니다.

## 🛠️ 사용 기술 및 프레임워크

- **Language**: Python 3.10+
- **Data Ingestion**: Kafka
- **Stream Processing**: Apache Spark (Structured Streaming)
- **Metadata & Storage**: PostgreSQL
- **Orchestration**: Apache Airflow
- **Visualization**: Grafana
- **Infrastructure**: Docker / Docker Compose
- **ETC**: OpenSky API
